Information for compiling and running the program.

Made by: Mikhail Vasilchenko, Filip Filiposki, Patrick Leonello.


To compile the program, run "g++ -Wall -Werror -std=c++14 -O -o ftt ftt.cpp Coin.cpp LinkedList.cpp Node.cpp MainMenu.cpp"

To run the program: "./ftt <foodfile> <coinsfile>", where foodfile is the name of the file containing foods, and coinsfile is the file containing coins.
